# Testes de API - FakeStore

Projeto de testes automatizados utilizando Postman.

## Tecnologias
- Postman
- JavaScript (Scripts de Testes)
- API FakeStore

## Testes implementados
- Validação de status code 200
- Validação de tempo de resposta
- Validação de estrutura do JSON
- Validação de propriedades obrigatórias

## Como executar
1. Importar a collection no Postman
2. Executar os requests
3. Verificar aba "Test Results"
